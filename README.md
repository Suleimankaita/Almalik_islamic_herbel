# 🌿 Almalik Herbal Center

A modern Inventory Management and Point of Sale (POS) system built specifically for Islamic Herbal Centers.

---

## 📖 Overview

Almalik Herbal Center is a full-stack web application that enables herbal store owners to efficiently manage inventory, sales, purchases, customers, suppliers, and financial reports from one centralized dashboard.

The project follows a **Monorepo Architecture**, making it easy to develop, maintain, and scale both frontend and backend together.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/almalik-herbal-center.git
```

## Install Dependencies

```bash
pnpm install
```

## Run Development

```bash
pnpm dev
```

Frontend

```
http://localhost:5173
```

Backend

```
http://localhost:5000
```
